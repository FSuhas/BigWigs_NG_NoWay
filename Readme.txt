# BigWigs

BigWigs est un addon de World of Warcraft pour prédire prédire certains comportements d'IA pour améliorer les performances des joueurs.

Cette version de BigWigs à été faite pour le Patch 1.12.1 et pour être utilisé sur le serveur NostalGeek.



## Comment l'installer

Copier le dossier BigWigs dans votre répertoire WoW/Interface/Addons.





## Support de langue

Ce BigWigs à été fait pour le client FR. Mais le client EN/US devrait pouvoir le faire fonctionner.
